Contoh Integrasi untuk Website Lain

üöÄ Cara Integrasi Sederhana

Untuk mengintegrasikan chatbot ini ke website Anda, tambahkan kode berikut:

1. Integrasi Basic (Copy-Paste)

<!-- Tambahkan sebelum closing tag </body> --> <script src="https://yoursite.com/cctv-chatbot-integration.js"></script> <script> CCTVChatbot.init({ position: 'bottom-right', // bottom-left, top-right, top-left theme: 'light', // light, dark, auto size: 'normal' // compact, normal, large }); </script>

2. Integrasi dengan Kustomisasi

<script> CCTVChatbot.init({ position: 'bottom-right', theme: 'auto', size: 'normal', features: { cameraLocations: true, emergencyContacts: true, maintenanceSchedule: true, operationalHours: true }, customization: { primaryColor: '#007bff', // Sesuaikan dengan brand Anda fontFamily: 'system-ui', borderRadius: '12px' }, integration: { apiEndpoint: 'https://api.yoursite.com/cctv', // Optional authToken: 'your-auth-token', // Jika diperlukan customData: { siteId: 'main-building', contactPerson: 'Admin CCTV' } } }); </script>

3. Integrasi dengan Event Callbacks

<script> CCTVChatbot.init({ position: 'bottom-right', theme: 'light', onOpen: function() { console.log('Chatbot dibuka'); // Analytics tracking, dll }, onClose: function() { console.log('Chatbot ditutup'); }, onMessage: function(message) { console.log('Pesan dikirim:', message); // Custom message handling } }); // API untuk kontrol programmatik // CCTVChatbot.show(); // Buka chatbot // CCTVChatbot.hide(); // Tutup chatbot // CCTVChatbot.sendCustomMessage("Halo!"); // Kirim pesan custom </script>

4. Integrasi untuk WordPress

<!-- Tambahkan di functions.php tema Anda --> function add_cctv_chatbot() { wp_enqueue_script('cctv-chatbot', get_template_directory_uri() . '/js/cctv-chatbot-integration.js', array(), '1.0.0', true); wp_add_inline_script('cctv-chatbot', ' CCTVChatbot.init({ position: "bottom-right", theme: "auto", customization: { primaryColor: "' . get_theme_mod('primary_color', '#007cba') . '" } }); '); } add_action('wp_enqueue_scripts', 'add_cctv_chatbot');
üìå Catatan Penting:
  • File cctv-chatbot-integration.js harus di-host di server Anda
  • Chatbot menggunakan browser TTS (Text-to-Speech) bawaan
  • Responsif untuk mobile dan desktop
  • Tidak memerlukan API key eksternal
  • Mudah dikustomisasi sesuai brand website
